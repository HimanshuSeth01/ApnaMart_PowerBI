# Power BI Sales Dashboard (India)

This project is an interactive Power BI dashboard built to analyze business performance across 28 Indian states. It focuses on key metrics such as Revenue, Profit, Average Order Value (AOV), and Quantity Sold.

## Features

* Tracks core KPIs: Revenue, Profit, AOV, Quantity Sold
* Includes dynamic filters for Quarter and State
* Uses DAX for monthly profit trend analysis
* Highlights loss-making months for quick identification
* Shows payment mode distribution (COD, UPI, Debit Card)
* Identifies top-performing product sub-categories

## Insights

* Helps identify profitable regions and categories
* Detects underperforming periods
* Supports data-driven decision making

## Tech Stack

* Power BI
* DAX
* Excel/CSV

## Project Structure


## Usage

Download the .pbix file and open it in Power BI Desktop to explore the dashboard.
